* Django 1.10


# Installation
To run the server:

    python manage.py migrate
    python manage.py runserver 0.0.0.0:8888

Now please check your site at http://localhost:8888/ddl

To get test cases:
    `unzip demo.zip`
Those templates should be enough for demo all functions.
